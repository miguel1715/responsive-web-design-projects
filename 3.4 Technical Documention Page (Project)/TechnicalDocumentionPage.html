Selectors

Selectors tell CSS which elements to style. They allow you to target elements by their type, class, id, attributes, and relationships inside the HTML structure.

Choosing the right selector makes your CSS cleaner, more reusable, and easier to maintain as your project grows.

  • Selector — targets elements by their tag name (e.g., p, h1, img)
    • p { color: blue; }
  • Class selector — targets elements with a class using a dot (.container)
    • .btn { padding: 1rem; }
  • ID selector — targets a unique element using a hashtag (#main)
    • #main-title { font-size: 2rem; }
  • Attribute selector — targets by attributes (e.g., input[type="text"])
    • input[type="text"] { border: 1px solid gray; }
  • Pseudo-classes — target states (:hover, :focus, :active)
    • a:hover { text-decoration: underline; }
  • Pseudo-elements — target parts of elements (::before, ::after)
    • p::before { content: "→ "; color: gray; }
  • Combinators — target elements based on relationships (descendants, siblings)
    • .nav > li { margin-bottom: 0.5rem; }
Box Model

The CSS Box Model describes how every element on a webpage is structured. Each element is treated as a box made up of content, padding, border, and margin.

Understanding the box model is essential for controlling spacing, layout, and alignment.

  • Box-sizing — controls whether padding/border count in the width
    • * { box-sizing: border-box; }
  • Content — the actual text or image inside the element
    • p { width: 200px; height: 100px; }
  • Padding — space between the content and the border
    • div { padding: 20px; }
  • Border — the line surrounding the padding and content
    • img { border: 2px solid black; }
  • Margin — space outside the border between elements
    • .card { margin: 1.5rem; }
